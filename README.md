```sh
git clone https://github.com/Florian1215/Push_Swap-Tester.git
```

***

### Evaluating test
```sh
python3 push_Swap-tester/push_swap.py evaluating
```

### Leaks test
```sh
python3 push_Swap-tester/push_swap.py leaks
```

python3 push_Swap-tester/push_swap.py 5 all
